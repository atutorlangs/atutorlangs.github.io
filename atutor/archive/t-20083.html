  • 2010-05-19 11:26:44

    Re: make assignment private

    The students might hand in the document into a group instead of an assignment.

    In "File Storage" -> "workspace" drop down list, there are 2 sections: Group & Assignment. The file submitted into a group is considered a group file and are shared by all students in this group, while the file submitted into an assignment is hidden from other students and can only be viewed and downloaded by the instructor.

    Re: make assignment private

    "4) Students can upload their files when they select the specific group"

    This action is to submit group files, which are intended to be shared by all group members.

    The correct way is, for instance, assignment A is added into group G, instead of selecting G in file storage to upload the file, students should upload the file into "My Files", then select the check box of the uploaded file and click "hand-in" button, follow the steps to submit it to assignment A.
