    pnATutor PostNuke Module

    Cas at http://www.nuy.info just wrote an enabling module entitled pnATutor that will allow sites to run ATutor within the main frame (block) of the PostNuke Content Management System.

    Usernames and passwords are passed from the PostNuke system to ATutor, so a double login is avoided.

    An adapted version of the include/html/user_bar_inc.php file is also inclluded that keeps users from jumping out of the portal when using items in the userbar.

    pnATutor Configuration

    I had some difficulty with the pnATutor module where I used a somewhat outdated copy, I think the download file was labeled 1.2. The one I have working still shows version 1.2 in the modules/pnATutor/pnversion.php, but the download/zipped file did NOT have any numbers in the file name and it was attached to a posting on the forums on Cas' site.

    The other problem I had was the ever present confusion over how to set up the linkage in PostNuke blocks. The default expectation would be that the module name is enclosed with [] (square brackets), but it will only work if it is enclosed in "curly brackets", ala {pnATutor} instead of [pnATutor]

    I have it working nicely on one of my sites and am willing to help via e-mail if that will help. My working site is www.dnlt.com. Look for the menu option under Site Features.

    Good luck or ask for more info.
