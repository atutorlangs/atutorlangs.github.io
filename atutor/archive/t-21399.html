    Re: Menu Navigation Rename

    You'll need to create language tokens (e.g. _AT('my_module') ), then translate them by inserting the language into the module.sql file.

    See the internationalisation section of the module developer doc.

    http://help.atutor.ca/developer/modules.html#localisation

    Once the module is installed, you can change the language through the admin's language manager.
