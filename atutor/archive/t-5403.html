  • 2005-08-17 08:09:31

    Turn on SMTP

    When using windows an SMTP server needs to be defined in the php.ini file. And, MAIL_USE_SMTP needs to be set to TRUE in ATutor's include/config.inc.php file.

    Create a php.ini file from the dist version and set the SMTP server in it.
