    Re: Semi admin Permissions

    There are only permissions to manage tests or not. There isn't a way distinguish between seeing marks and editing tests.

    All permissions are set in the course enrollment manager. Experiment with different permissions to see if it will give you what you need. Permissions are set course by course. You can't give access to all tests etc. with one setting.
