    Need a session

    You are bounced to the login page if there is no session running. Use the bounce.php page to create direct links to courses. You can copy the bounce link to a course from the Browse Courses list. You can not create a direct link to a course content page. Only to the course home page.

    eg
    http://www.myserver.com/atutor/bounce.php?course=46

    Increasing font sizes is generally a browser function. If you really need to change the font size adjust the "select.dropdown" style in the ATutor stylesheet (stylesheet.css in the ATutor root directory). Add something like "font-size: 1.2em"

    Public Courses Only

    You can only use the bounce page to link to public courses. You would have to login to a protected or private course normally.

    In public courses, you can only link to the course home page. Not to content pages.
