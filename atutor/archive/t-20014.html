    Re: I can't find assignment specific tests for groups

    First you should create a group (Manage -> Groups). After then you'll be able to see that group while editing a test (Manage -> Tests & Surveys -> [i]select a test[/i] -> Edit) and, accordingly, assign the test to that group.
