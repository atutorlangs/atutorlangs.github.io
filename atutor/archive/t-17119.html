    It looks like you found a bug. There is a extra "../" in the path to the emoticon images from TinyMCE. I've reported it to the bug tracker, and it will get fixed in due time.

    In the meantime, you can edit the announcements manually, with the TinyMCE editor turned off, and adjust the paths by removing one "../".
