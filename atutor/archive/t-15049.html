    I'm assuming you're using the [media][/media] tags. If not, search for "media" in the Instructor Handbook.

    The player that opens should be the one set as the browser's audio player. You may need to adjust this setting. If you're playing an mp3 for instance, look in the browser's preference settings where the associations are made between the mp3 file type, and the application they should opened in. Wording of these options will vary from browser to browser.
