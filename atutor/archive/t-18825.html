    Re: Lost Admin Password

    Hi Charles,

    Do you have access to the ATutor's database via phpMyAdmin or a similar tool?

    If you do, then go to the AT_admins table, find the row with your login name, and set new password.
    You'll need to select "SHA1" from the dropdown box right to the left from the field where you enter your new password.
