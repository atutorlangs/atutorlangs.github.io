    Re: Install Acollab or Atutor first

    ATutor 1.5.3.2+ now contains all the functionality of ACollab, except for the Calendar, which will likely be available in the next release.

    Generally ATutor installs before ACollab, but it does not really matter.


    Question:
    1) Should I download Acollab or use the option under install module in Atutor once logged in as administrator.

    [green]Use both. Install ACollab, then install the module. The module just links the standalone version of ACollab into ATutor [/green]

    2) Should I uninstall Atutor, download acollab and install and then install Atutor as an add on.

    [green]ACollab can be used as an addon. Not the other way around [/green]

    Unless you need the ACollab Calendar, I would suggest not installing ACollab.
