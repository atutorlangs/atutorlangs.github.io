    Re: Test problem

    This sounds like a timeout problem that was present in ATutor 1.6. Time out is set to 10 hours in earlier version.

    If it is in fact 1.6 you are using, install the patcher module, and install the appropriate patch it lists to correct the timeout problem.

    If you have another ATutor 1.6 installation on the same server as 1.5.4, it may be causing the problem. Run the patch on it, to correct the problem in 1.5.4.
