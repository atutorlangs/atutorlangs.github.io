    Re: Students' results - Restart tests

    1. There are a few ways to view results. One way, in the test manager select a test, then click the Submissions button, choose a submission then click the view button.

    2. You probably need to set test questions to appear 1 at a time. You can't restart a test that has all questions displayed on one page.
