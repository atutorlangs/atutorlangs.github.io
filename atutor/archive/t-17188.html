  • 2009-04-21 01:14:20

    Re: Content share between courses

    Hi,

    You can copy it using backups (Manage -> Backup). The steps to do so:
    1. Create a backup in the course that already has the needed content in it.
    2. Download the backup to your computer.
    3. Go to the target course.
    4. Upload the backup there (via Backups, not File Manager).
    5. Restore the backup (partially, by selecting only those items you want to restore).

    The same also works for tests and other items.
