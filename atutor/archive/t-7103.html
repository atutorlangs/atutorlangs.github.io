    Hi, on my IIS server also problems came up in the final stage of the install process, due to include/config.inc.php being read-only. What I did:

    -download include/config.inc.php with FTP to a local computer
    -edit the file with a text editor and put in the right values
    -upload the file to my ATutor server (after first removing the read-only attribute of the original file so it can be written over).

    Hope that helps.
