  • 2005-09-13 10:40:23

    Paste from file

    Because your HTML files are not assembled in a standardized format (e.g SCORM or IMS packages) you will need to import them one-by-one. You can use the paste from file feature in the content editor to pull the HTML files into ATutor. You will also need to upload the images etc. that link into those files.

    Once you have all your files recreated in ATutor, then you will be able to package them in a standard format and move them around easily, or move them into other standardized systems.
