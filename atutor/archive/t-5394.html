    Side Menu boxes

    An instructor can set which boxes show up in the side menu by going into a course > Manage > Student Tools > Side Menu. Choose the empty choice if you wish to remove a box.

    If you want to remove them entirely from your theme, edit the theme/[theme name]/include/footer.tmpl.php file and remove the appropriate code.
