    Re: restoring a backed up course to a new server

    The course title is not restored, but you can overwrite your current course content with a backup.

    If you want to create a new course with those backup files, then you need a new course title.

    Unfortunately, the backup doesn't export an image of the course. Course settings will be gone, in which they will be back to the default settings. The backup will only backup the checked items within backup->create.

    Hope this helps.
