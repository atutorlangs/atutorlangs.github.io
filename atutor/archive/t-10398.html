    Re: Tranfer data between atutor versions

    You can try making copies of the ATutor directory, and of the mysql data directores for ATutor, and moving them over to the new server.

    Then run the ATutor installer in upgrade mode.

    There are other ways as well, restoring backups of courses from the old to new server.

    Exporting the database with phpMyAdmin, and restoring it into the new mysql. And copying the ATutor directory.
