    Not a theme

    The atutor.ca site is not using ATutor as its CMS. It is a custom site, so no, you could not get Projects on your ATutor site. You could create a theme that looked like atutor.ca, but since Projects is not a function of ATutor, you would not be able to include it as part of the theme.
