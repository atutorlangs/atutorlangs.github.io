    Re: LMS Standard

    You probably want to look at the IMS interoperability standards.

    http://www.imsglobal.org/specifications.html

    As well as W3C markup and accessibility standards

    http://www.w3.org/TR/WCAG/
    http://validator.w3.org/

    Here's a list of the standard ATutor complies with:
    http://atutor.ca/atutor/

    As for supported files, you can add just about any type of file you might use as content. There are restrictions on executable type files, for security reasons, though if your client really wanted to use them, the restriction can be turned off.
