    Re: The Registration page become blank after server changed from windows to linux

    Its difficult to say what the problem might be.

    Case sensitivity could be the problem. It appears to be a customized version of the registration form, passing through the registration_sel.php file (which is not an ATutor file) before linking to the registration.php file, which I'm assuming you have modified.

    You might have a look at the php error log, to see if there is anything there that might suggest what the problem is.

    Also check which version of php you are using, there have been some issues with 5.3.0+ with version of ATutor prior to 1.6.4. In which case you may need to upgrade ATutor, or revert back to an older version of php (<= 5.2.9)
