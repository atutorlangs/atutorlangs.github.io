  • 2004-03-18 17:26:14

    SCORM content, not SCOs

    ATutor does not have a SCORM runtime environment yet, so it can't play SCOs. You can only import/export IMS/SCORM Content Packages.

    You could upload your SCOs, link them into your course somewhere and make a player like Reload available, then have your students download them for offline viewing.

    Or, send us a package to look at, noting what it was created with, or directions to the content onlne, with instructor access.

  • 2004-03-26 07:35:36

    Set DirectoryIndex

    It sounds like your web server needs to be set to recognize index.php as an index page. In apache this is done by adding index.php to the DirectoryIndex variable in httpd.conf, like so:

    DirectoryIndex index.php index.html
