    Customizing ATutor

  • 2003-12-01 12:40:18

    They are not yet options

    If they are hard coded, they are not yet options.

    For now, any changes you make to the code will have to be redone when you upgrade, unless you make your change available for us to add to the next distribution. If you want to remove the registration link for instance, it must be an admin setting, and/or a course properties setting.

    You could always try submitting a feature request through the approriate forum. If others also find your request desirable, we will add it ourselves. The more people requesting, the more likely we are to add the feature.
