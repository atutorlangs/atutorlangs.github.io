    Webservers and PHP rather OSs

    There have been reports on many UNIX based systems. Issue generally arise with regard to webservers and versions of PHP, than with OSs. ATutor seems to run equally well on current versions of *NIX and Windows.

    Apache 1.3 is the Webserver of choice for PHP apps. A few IIS user have reported minor issues associated with permission setting. We've had no reports from AOL or Zeus users that I recall. ATutor does not run well on Apache2 systems, though that's likely because PHP support for Apache 2 still has some bugs to work out. You can search the ATutor site, and the forums, for references to other webservers...
