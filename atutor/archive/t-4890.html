    Upgrade from 15RC1 to 15RC2

    Is there a simple way to upgrade from ATutor 1.5RC1 to RC2?

    The upgrade process during the installation does not work for release candidates. To try out, I installed as a fresh installation over my existing one (i.e. same DB, same content directory). Then I'm preparing to update my custom theme to reflect the changes. Is this correct or should I totally reinstall everything?


  • 2005-06-20 18:08:06

    Copy RC1 over to RC2

    There were no changes to the database between RC1 and RC2 so you can save the include/config.inc.php file, the content, themes, and the sco directories (if you created a sco directory), unpack the RC2, and copy the file and directories over to the new installation. You can not run the upgrade script on RCs, as you indicate.
