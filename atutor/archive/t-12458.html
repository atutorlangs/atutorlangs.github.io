    hi jmead,

    this is what I would have done:

    1. create the course without images in the description first.
    2. go to a content page in the course (under content navigation), edit it, open the file manager and upload the images there.
    3. insert them into your content page with the "insert" button (NOT using the visual editor).

    this will give you the IMG SRC code for the pictures, copy that code, or only the URL (the content of SRC=""), go to magage -> properties again, where you can use the code you just copied to refer to your images.
