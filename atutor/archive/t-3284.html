    Unzip and import via phpMyAdmin

    You can unzip the file and import the CSV file it contains into the lang2 table of ATutor, using an application like phpMyAdmin.

    zlib is supported by default in current php distributions, Try upgrading your system if you are using an older version of php, or recompile php with zlib support if you want to continue using an older version.
