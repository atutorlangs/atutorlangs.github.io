    Reinstall if necessary, to set the proper permissions on the directory.

    Apart from using Apache 2, I don't see anything that might be causing the problem. Language packs import into 1.5 without any trouble on our systems. Make sure the content/ directory, and the import/ directory within it, are both writable. View the properties of the directories and uncheck Read Only if it is checked. Create the import directory if it is not created automatically when you try to import a language.
