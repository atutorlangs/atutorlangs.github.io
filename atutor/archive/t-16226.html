    Re: Error when uploading pictures to lab instructors

    Yes, this is a permission error. Does the directory exist to upload picture to, and is it writeable and owned by the Web server user (e.g. "nobody")

    If this is something that has started recently, you may want to check with your provider so see if anything has changed recently, such as an upgrade to the system, or perhaps being moved to another server.
