    !.4.3 pack as a base for 1.5

    Yes you can use 1.4.3 packs as a base for 1.5. Before you upgrade, export the language and save it locally. After you've upgrade import it back into your 1.5 system. Then translate as you normally would. Assuming the English translation for 1.5 is intact, the langauge manager will identified the new and updated items that need to be modified in your old 1.4.3 translation.

    Better though, using your translator account, login to the atutor.ca translation site and download the partial language pack, which may already include some of the required changes to update the pack for 1.5.

    And, better than that, login to the atutor.ca translator site and translate directory into the primary ATutor language database.

    The main translator documentation is available at:
    http://atutor.ca/atutor/docs/translate.php
