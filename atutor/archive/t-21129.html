    Re: roles

    From the level of the whole ATutor, there are only 3 type of users: admin, instructor, student.

    Instead, instructor of the course can assign privileges to students including managing the enrolment of this course.

    Not sure if this feature satisfies your need. If you are interested:

    Login as a instructor -> go in a course -> tab "Manage" -> Enrolment -> select a student -> button "privilege".
