    Edit Apache httpd.conf

    You are missing index.php as a value for DirectoryIndex in Apache's httpd.conf

    e.g.
    DirectoryIndex index.php index.html

    Double check your MySQL user/pass. Typing it wrong is usually what that eror message usually means.
