    [SOLVED] Import of courses from ATutor 1.6.2

    Hello,

    I discovered that teacher/instructor was importing backup file instead of ims.zip file and that order of pages was disturbed. THus, it is not bug.
    But, I recommend one thing:
    It may be good to add in backup script that adds to backup .zip file word backup as it adds to exported course ims. Thus, it will be easier to differentiate between backup name.zip and export ims.zip files.
    By the way, it adds numbering before the title of pages although that numbering did not exist.
