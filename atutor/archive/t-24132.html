  • 2013-06-15 14:41:03

    Re: information not updating using Editor

    All language is in the database.

    The other way of modifying language is to use the translator tool. It becomes available after setting translate mode on manually by editing the include/vitals.inc.php file near the top.


    define('AT_DEVEL_TRANSLATE', 1);

    See the Translate tab (next to the Editor tab you've been using) for more details on using the translator.
