    Tweak the language

    The language might have a bug in it. Open the page_info template variable with the administrator's Language Manager, and check the associated language for errors. If there are any blank spaces after the end of the text delete them, then save your changes. Or, try retyping the string. Or, removing it alltogether. Or, deleting then reinstalling the language.
