    Disabeling functionality

    I have a general question concerning installation and configuration.

    Is it easy or difficult to disable functionality in Atutor?

    I would like to set up a LMS starting with little functionality as a system that support the activities of the teachers in the physical classroom. As the users become more proficient I would like to introduce more functionality.

    It seems to me that the biggest problem would be to have the teachers and students to use the system.


  • 2006-04-02 10:11:08

    Re: Disabeling functionality

    Very easy to disable functionality via Manage>>Student Tools.

    You can also enable and disable modules, and choose which features are available in a newly created course through the Admin>>Modules feature.
