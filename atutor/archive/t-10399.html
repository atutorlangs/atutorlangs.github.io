    Re: customising header

    you would have to edit the files:

    atutor/themes/[theme name]/include/header.tmpl.php for the html in the header, and probably

    atutor/themes/[theme name]/styles.css for the CSS.

    if you have enabled more than one theme you will have to customize these files for every theme (as of now you're using the Blumin theme it seems)
