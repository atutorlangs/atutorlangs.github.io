    Re: disabling course list feature in Instructor interface

    it's right that all instructors can add users to a course. in addition they can create courses and manage course properties.

    an alternative approach for you could be to use assistants instead of instructors, as assistants can be fine tuned more to what they have the privilege to do. for example you could deny them the enrollment privilege.

    a side effect would be that they can not create courses or edit course properties, so this would also have to be done by admin.
