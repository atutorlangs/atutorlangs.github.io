  • 2005-04-18 19:14:38

    Use the backups tool

    See the Instructor Tools, then Backup Manager.

    Create a backup. Download it from site #1. Go to the Backup Manager on site #2. Upload the site #1 backup to site #2. Restore the backup on site #2.

    You can also move courses this way as the admin user.

    See the HowTo course for details:
    http://www.atutor.ca/atutor/docs/howto.php
