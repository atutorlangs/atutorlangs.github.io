    Re: Can't generate new content

    You could restore the backup you created before upgrading, though you'd loose anything new added since then.

    I should ask, are you using a custom theme, or what theme from the themes include with ATutor are you using?

    The other possibility, though it would be a lot of work, is to examine the upgrade scripts in install/db/ since 1.5.5, and particularly the changes to the content table, and perhaps just adjust that table. Problem is though there would no longer be an upgrade path if you continue using a tweaked 1.6.3 db for 1.5.5.

    Another option might be to install a fresh copy of ATutor, then take backups from the old version and move them over to the new version and use it as your production system.
