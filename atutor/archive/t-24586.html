    Re: Moving Courses From version 1.6 server to new version 2.1.1 server

    The best option is to upgrade from 1.6 to 2.1.1 if you are able to copy the entire ATutor 1.6 to the new server. Otherwise backups to the way to go. It should work fine.

    There have been a variety of issue reported with PHP 5.3.3, so that may be the problem. If you have an opportunity to upgrade to PHP 5.4.*, it may fix the problem.

    Otherwise try creating a backup on the old server and restoring it there to determine if its the creating or restoring that is failing.
