    Create a module?

    I'm not familiar with Invision, so I can only guess that you might be able to create a module of it for ATutor. You will not be able to move your current users into ATutor, but you should be able to allow new users to register in ATutor, and use their ATutor session to access an Invision module (strictly guessing).

    ATutor does not store md5 passwords, nor do I think it will ever. Password encrytion is on the todo list, but we will likely use SHA1 instead.

    See the Module Developer documentation for more about creating modules.
