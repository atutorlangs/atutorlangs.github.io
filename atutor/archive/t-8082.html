    Re: Accessibility features help

    This is a problem with the browser, not ATutor. ATutor follows the standards as they are intended to be implemented. Browser vary in their compliance with the standards, so some may not be able to take advantage of various accessibility features. There is nothing that can be done to ATutor to workaround these problems.

    Firefox should be fine. It is fairly compliant with the standards. IE is also good with the standards. I am not familiar with Netcaptor, but I am guessing it uses non-standard menu accesskeys.
