    Re: Importing and uploading errors

    Set both of these values in php.ini to 30M (500M may be too large)

    upload_max_filesize
    post_max_size

    Then as admin under system preference set the system default file size, in bytes.

    After that you can control course by course file sizes under the admin Courses tab, by clicking Edit
