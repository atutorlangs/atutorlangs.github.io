  • 2008-07-25 03:52:39

    Re: Groups

    Pretty well everything except content (tests, forums etc.) can be assigned to groups separately.

    However, if you do need to give them separate content, the only option I see is creating several courses and using backups for moving that content along with tests and everything else between them.
    Not so elegant, but usable unless you have too much students.
