    Re: Help with Content Directory - First timer

    The content directory is where all your data goes (except the data that will be stored in the database).

    You simply need to create an empty directory on your server and tell ATutor where it is. Since it will contain your personal data it is best to create it outside the /www directory.

    You can create the content directory via shell access to your server or by using a FTP client like Filezilla.

    On my hosted server ATutor is located in :
    /home/3/a/acme/www/ATutor/

    I created the content dir outside my /www directory like this:

    /home/3/a/acme/content/
