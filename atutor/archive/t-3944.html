    Content Packages only

    The zip files have to be a standard IMS/SCORM content package. You can't import other zip files, though you can upload them through the file manager, and unzip them there.

    Subject: ERROR: Import Cancelled Successfully

    Found the answer for anyone having the same problem: File was larger than reported. Set the max file, class ...etc. larger and it loaded ok.
