    Help us Help You

    The more information you can provide about your ATutor installation, the easier it will be for us to help, and the faster you are likely to find a solution.

    You will want to email any access information to us directly, rather than posting it to the forums.

    [Since we don't publish raw email addresses anymore, because of all the spam harvesting, you'll need to decipher info/AT/atutor.ca if you want to reach us directly. Do not send your support requests to this email address, only your access information. ]

    Here is a list of things you can describe with your help request.
    1. The version of ATutor you are referring to.</li>

    2. The operating system ATutor is installed on (Windows, Unix etc., Mac ...), and its version

    3. The Web server you are using and its version (Apache, Netscape, IIS...).

    4. The version of PHP and MySQL being used (php 4.2.0+ and MySql 3.23+ are required) .

    5. The Web browser you are using to view ATutor, it's version number, and any default settings you may have changed.

    6. Do you have root access to your server, or are you a hosted user on someone else's server, etc?

    7. List the steps you took prior to experiencing the problem.

    ACCESS INFORMATION
    8. Provide access information to your ATutor installation with appropriate admin/instructor accounts.

    9. Provide a URL to a phpinfo page so we can see how php has been installed on your system.

  • 2008-01-23 08:49:54

    Re: Installation Issue - Help Required

    See the requirements for a list of things required before installing ATutor.

    http://www.atutor.ca/atutor/docs/requirements.php
    I'd guess MySQL is pointing to a port other than the default 3306. You can change the port in the include/config.inc.php file of ATutor.

    ATutor will not work with IIS 5. There have been a few reports of it working with IIS 6, but we recommend using Apache as your Web server. You can install a package like XAMP on windows, which contains everything you need to run ATutor.
