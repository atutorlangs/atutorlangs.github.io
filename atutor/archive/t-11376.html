  • 2007-08-16 01:58:51

    Re: Import questions previusly exported by ATutor

    To do that, you should create a backup in the first course and upload it to the course you wish to copy the questions to.

    Then you should select the backup and click [i]Restore[/i] button. On next page check only [i]Test & Surveys[/i] and then [i]Restore[/i].

    Be careful when selecting appropriate Action as it can delete all existing material.

    Re: Import questions previusly exported by ATutor

    The IMS question export utility in the current version is experimental only, since no other systems yet supported the IMS QTI 2.1 standard. We will add the import tool some time in the future. IndieRect is correct. For now you can export a course backup, then import it into a new course and restore the questions contained in the backup.
