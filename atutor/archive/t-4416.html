    How can I add a power point presentation?

    In Powerpoint save you presentation as HTML

    zip the html files to a single file and upload into the ATutor file manager

    In the filemanager unzip the files

    open the content editor then the file manager linked from the content editor

    Click Insert link in the file manager next to the first slide to create a link in the content editor area

    Save the contents of the content editor and close

    Click on the link you created to open the power point presentation
