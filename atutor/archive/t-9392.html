  • 2006-12-01 08:30:08

    Re: scorm file importation...help!

    Be sure the ATutor Scorm Packages module is installed. See Admin>Modules>Install Modules

    Make sure the sco directory has been created, and that it is writable by your web server. Enabled the Packages tools for a course and use it to import the zipped SCO. Do not try to unzip the SCO yourself.
