    FAQ clarification

    In FAQ 4. I get "Fatal error: Call to undefined function: cache()" when I start ATutor for the first time. Why?

    You may need to create a directory that is world writable, or use an existing one (often /tmp on Unix, or C:\windows\temp on Windows), where ATutor pages can be cached. See the Configuration options for more about the cache directory. If all else fails, comment out the cache configuration line in the config.inc.php file to turn caching off.
