    Re: can't find Instructor Tools

    Hi!

    There was a major redesign of ATutor in version 1.5, so the 1.4.3 documentation you are using is obsolete now.
    You should use "The ATutor Handbook" instead. It is installed with ATutor and is also available at http://help.atutor.ca .

    Course Default Preferences tool really existed in 1.4.3. In 1.5 it has been replaced with Manage -> Student Tools.
    Though you will find that the ATutor behavior has changed and some settings described in 1.4.3 docs are simply absent in the new ATutor.
