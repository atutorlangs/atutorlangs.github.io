    Re: Creating Subtopics

    Hello ebusinesstutor,

    I think you need to add a container for sub page content from the side bar (can't see where else you can do this from)

    In the side panel navigation (enable it if you've turned it off/it isn't already visible) you should see the "Content Navigation" panel; to the top right of this panel are 3 small icons "Add Top Folder", "Add Top Page" & "Enter Edit Mode".

    "Add Top Folder" will create a folder (as you might expect). You can then move top level pages into the sub section(s) you have just created in the "Arrange Content" tab.

    Hope this helps
