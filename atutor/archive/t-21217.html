    Re: reports

    1) A glance at learners' names in a course:
    Login as instructor -> "Manage" tab -> "Enrolment" link

    Don't have report "total time spent on course", "times logged in", "tests received" is in 3)

    2) Specific learner report: courses assigned, last logged in
    Login as admin -> "Users" tab

    3) test reports
    Login as the instructor -> "Manage" tab -> "tests and surveys" link

    For more info: http://help.atutor.ca/index/index.php
