  • 2010-06-14 08:49:15

    Re: Test error

    Check to be sure the tests have questions associated with them.

    Try restoring a backup of the course, and if that does not work, try exporting the test from the parent course as a QTI package (assuming you are using 1.6.2+)

    If niether of those works, there may have been a problem with the install. Try a fresh install, then import a backup or QTI package again.
