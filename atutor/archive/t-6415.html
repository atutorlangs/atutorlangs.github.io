    modify php.ini

    The php configuration file (php.ini) controls the allowable upload file size. You will need to edit it, then as the ATutor administrator modify the course properties to allow 54M+ files to be uploaded.

    see:
    http://atutor.ca/view/9/2415/1.html?reply=2416

    Or, you can use the URL import feature, which is not restricted by upload file size.
