    Full Fledged ECommerce Mods Beta Testing

    First of all, thanks to the developer of the ecomm module and greg. I wouldn't have worked on this if I haven't got inspiration from them.

    This is a Full Fledged E-Commerce module which can be used for a variety of purposes. As I am still improving it, I'm releasing it for BETA testing. As of now, the following is the feature list:
    [b]
    Courses now have a validity - Right from One Week validity to a One Year Validity - a total of 7 options - DONE
    Multiple Cost Association using Ajax - Separate fields for entering costs for a One Week Course, Two Weeks Course and so on up to One Year Course - DONE
    Instructor can create Course Combos - Offer discounted packages to students if they want to buy more than one course - DONE
    The user will be given an option to enter the validity dates for different courses incase of a Combo Course - DONE
    After the validity date is over, the student is unenrolled automatically - DONE
    The user will be able to pay the money over PayPal IPN - DONE
    Administrator interface for a confiduring the mod - DONE
    Multi Language Support - Almost Done
    Multiple Payment System - As of now, PayPal is supported provided you have a Business Account. I've tested using Sandbox Environment
    Documentation - PENDING
    User will be displayed even the minute information about his course - Course Start Date, Course End Date etc. - DONE - For this, you will have to replace the original index.tmpl.php file in the themes\default\users directory with the one i provided in the rar file. This is very important for the proper functioning of the mod!
    Email notifications - PENDING
    [/b]

    ==============================

    [b]
    Administrators:
    Can configure the module

    Instructors:
    Can create Course Combos and Assign costs to them
    Can assign costs to Individual courses

    Students:
    Can join Individual courses
    Can join Combo Courses

    PS: I'm still new to ATutor so you might be finding some trivial flaws. Let me know and I'll rectify them. There are more features but I'll write them down when I remember them ;)
