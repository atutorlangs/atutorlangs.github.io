    Re: Can I create an style (CSS) for a particular course?

    You have a few options:

    You can customize a theme, create a category and assign it to that category, then add your course to that category. (see category themes in the handbook)

    You can insert styles in the head area of each content page through the content editor.

    You can create a custom stylesheet and place it in the course content directory via the file manager, then use a link element in the head area of each page to link to it.
