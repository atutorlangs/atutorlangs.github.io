    Authoring tools

    your server is running PHP in safe_mode, which means that a lot of functionality is being limited if you do not make some adjustments to the installation.

    You should disable safe_mode in your php.ini file or change the ownership of the content directories to "nobody".

    We're still trying to deal with the safe_mode issue, hopefully an upcoming release will be more safe_mode friendly.
