    There have been similar problems reported by others using IIS. IIS support for PHP leaves lot to be desired. If you are planning on setting up a production installation of ATutor, you should probably be using Apache 1.3.

    We do not have an IIS webserver available to reproduce the problems, so there's not much help we can offer.
