    Re: ATutor content with php

    You would not be able to embed PHP in your content. Javascript you can add into the Head area in the content editor.

    If you really need to use PHP, you could create a module, or you might be able to create separate PHP files, disabled the restriction on PHP uploading in the system preference, then upload those files using the file manager. Allowing PHP uploads will create a security vulnerability, so be sure to reenable the restriction once your files are uploaded.
