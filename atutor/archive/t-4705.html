    I'm guessing you need to turn register_globals off to fix the first problem. Use the search to find answers. (terms php_flag or register_globals)

    The HowTo documentation is a course. You can upload the backup with the Instructors Backup tool then restore it, or you can import the content packages using the Instructor's Content Packaging tool. All of this is described in the HowTo course, which you can browse on atutor.ca
