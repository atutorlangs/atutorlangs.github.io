    Re: Links between Content

    Linking between content pages it not a good practice from an interoperability perspective. If you duplicate the content through an export or course backup, all those links will break.

    You can use the related content feature in the content editor, which generates links in a side menu block.

    The Sitemap module might work, if you have not discovered it already.
