    The Announcements page

    The announcements page is the course home page.

    There is no so called ATutor Home Page though I suppose you could call the login screen the home page.

    There is no "default" course, though it would not be difficult to copy the login screen and have it open a default course

    With little imagination, and HTML skills, you can create just about anything.

    Please see the ATutor HowTo course.
