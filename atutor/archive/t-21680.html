    Re: Remove Tests from Content Navigation

    There isn't a way to be absolutely sure a student has read content pages before a test becomes available.

    I'd suggest removing the test associated via the content editor, to remove it from the menu, then manually copy the link to a test, right clicking on the test's name in Tests & Surveys, then paste that link into the last content page of the section as part of the content.
