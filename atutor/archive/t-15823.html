    ATutor should have no trouble connecting to MySQL if the location, login, and password for it are correct. Double check the login information for mysql. Please be certain it is correct before we start investigating further.

    Also be sure your mysql user has permission to create databases if one does not already exist, and that it has permission to create and alter tables in the database. Ask you service provider if you're not sure.
