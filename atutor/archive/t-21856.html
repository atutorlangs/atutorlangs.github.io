    Re: Captivate 5 SCORM IMS packages

    Admin tools are part of the system, not a special install.

    If you are logged in as the admin, you should see a "modules" tab. If not, then you are not logged in as admin, or you are a secondary admin that does not have module privileges.

    Did you install this ATutor system, or are you an instructor? Setup of the admin should have been pretty obvious during the install.

    The SCORM module is not ours, and it is not being maintained, so it will not be included with the core code. Now-a-days IMS common cartridge is the way to go. SCORM content is too restrictive for most purposes. It's available as an addon, for anyone who needs it. Our efforts though are focused on developing for the more flexible IMS common cartridge standard.
