    Upgrade, then transfer

    I'd suggest upgrading your old system to ATutor 1.4 first. Then copy the members table to ATutor 1.4 on your new system.

    Or, you can transfer a basic user list by exporting then importing through the Enrollment Manager.

    Language gets replaced when upgrading, so you should document changes you've made so you can reapply them after an upgrade. The lang2 table contains language other than English, and is emptied when upgrading. You would need to download and install a new language pack in this case. The lang_base table contain the English base language.

    The preferences table no longer exists in ATutor 1.4. Preferences are now stored in the members table.
