    PHP not compatible

    Its PHP that is not fully compatible with Apache 2, rather than ATutor

    PHP is fully compatible with Apache 1.3.

    You can still run ATutor on an Apache 2 system, though you will likely run into problems in places. There are a number of production sites setup using Apache 2, though some ATutor features were removed. If you plan on setting up a production system, you are strongly advised to use Apache 1.3.
