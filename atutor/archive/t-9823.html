    Re: XML error: Undeclared entity error at line 13

    The issue is caused by the & character in the glossary.xml file. You could either delete that file then re-package.

    The issue is with the exporter (which does not escape the & correctly in the glossary term) and in the importer (which does not correctly link glossary terms).

    This issue has been added to our bug tracker and a fix will be available with the next release.
