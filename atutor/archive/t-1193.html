    Set your personal prefs as course default

    Set up the course preferences as your personal preference settings first, then apply them to the course by clicking "Save and apply these preferences to This Course". You can go back and change your personal settings later without affecting the course defaults. See the HowTo course for more about prefs...

    Try it, then ask questions....

  • 2004-03-03 16:11:56

    Default prefs broken

    The ATutor default prefs were changed in 1.3.2 in error. We'll change them back in the next release.

    The override preference was turned off in the ATutor Default preference settings, which are applied for new users.

    You can fix the problem by logging into MySQL and adjusting theme 4 in the theme_settings table, from "0" to "1".

    It should look like this:
    s:13:"PREF_OVERRIDE";i:1;

    Be careful not to break the rest of the preference string...

    Copy prefs from courses table

    There seems to be other issues with setting course default preferences, that we can't deal with right now. We'll try to have this fixed for the next release (1.3.4).

    For now set your course default preferences as you normally would. Then copy the prefs string that gets saved in the courses table, for the course you saved defaults for, into the theme_setting table for theme 4.

    This will be the default preferences for your system.
