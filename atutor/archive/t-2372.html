  • 2004-09-22 14:05:32

    Discovered the problem ... probably

    While adding more files in content/ I reached a point where the problem appeared again. Browsing the php.ini configuration file found the parameter

    memory_limit = 8M ; Maximum amount of memory a script may consume (8MB)

    in the


    ; Resource Limits ;

    section.

    Once put to something higher (tried with 10 MB) the Export Backup worked again.
