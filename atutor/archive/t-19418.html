    It's difficult to say what the problem is. Not all SCORM authoring tools are created equal. If you can export an IMS Content Package, try using it instead.

    And, don't confuse content packages with sharable content objects. The latter requires the SCORM Package module to be installed, while the former can be imported directory into ATutor.
