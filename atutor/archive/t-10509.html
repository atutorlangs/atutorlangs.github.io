    Re: Create Category link for questions

    No, on course the prerequisite.

    You will want to edit registration.php in the ATutor root directory to add required fields.

    You could add a code of ethics field as a required field in the registration form. The form itself is part of a theme, so fields can be customized there.

    Also keep in mind that custom changes such as these will be lost at upgrade time, so you may want to document what you have done. Another option might be to create your own registration module to replace the current one. The module could then be reinstalled on an upgraded system
