    I used absolute path

    What I did was this:

    Upload the image file using File Manager.
    From File Manager, right click on file link , and from menu, choose Properties.
    Then copy the the URL (absolute path).

    Then I "visual-edit" the page, click the Insert Image icon, and paste in the absolute path.
