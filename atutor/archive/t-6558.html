    A research project

    SCORM is essentially a standard way of defining the relationships between a set off content object using an XML file. Of course its more complex than that. Giving you the full story here goes a little beyond the type of support we can afford to offer.

    There is loads of information out there about SCORM if you want to do the research.

    No SCORM does not allow you to use others test (though as part of a sharable content object SCO I suppose yes it does). The IMS QTI standard is used to define interoperable questions and test. QTI is not supported in ATutor yet, but it is high on our list of things to do.
