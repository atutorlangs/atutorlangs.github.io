    Re: can't import questions

    You'll need to export and import a course backup to import the questions from a previous version of ATutor.

    IMS test packaging in previous versions was an experimental version of the QTI 2.1 standard, which no one is adopting. In the current version of ATutor you can import and export IMS 1.2 questions and tests.

    Bottom line, you've exported a QTI 2.1 package from your previous version. There isn't a QTI 2.1 import utility available yet.
