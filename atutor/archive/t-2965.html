    Language Manager

    Login as the administrator of your ATutor installation and use the Language Manager to modify language. If language is missing, which appears to be the case for [discussions], you will need to add a new entry into the language database (the lang2 table) using an application like phpMyAdmin.

    Make sure you are using the correct language pack for the version of ATutor you are using.
