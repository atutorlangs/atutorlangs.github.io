    Re: Allow students to upload files in

    If you mean File Manager, you can give students file manager privileges by slecting the student and pressing the Privileges button in the Enrollment manager.

    File storage is for file sharing between members of a course, File Manager is for managing course specific files.

    Re: Allow students to upload files in

    Students can't upload to the course files unless you give them File Manager privileges, which could be dangerous as they can also delete files. This provilege is typically given to teaching assistants or co-instructors.


    You probably want to create a group (Manage>Groups), perhaps adding all the students to it, and make the File Storage tool available to the group.
