    No, you can import the 1.6 language pack into your 1.6.1 system via admin's System Preferences -> Languages page (enable translation mode first).
    If my memory serves me right, one never could import an old langpack unless for a translation.

    Also, there always remains an ability to tweak 1.6 lang pack (or ATutor source code) so that the system accepts it as 1.6.1. :D
