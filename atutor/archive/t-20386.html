    Re: upgradeing to 2.0 and moveing to new server advice needed.

    Option 2 may be a better bet if its just user account info that needs to be transferred. You'll probably want to manually export the members table from the old site, and importing directly into MySQL using something like phpMyAdmin.

    If you are using a custom theme, it will need to be recreated.

    Modules should work fine in 2.0, though these are third party modules. We have not tested them.
