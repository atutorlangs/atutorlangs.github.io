  • 2009-08-26 15:39:50

    Re: Importing SCORM questions

    The module, created by someone else, is not setup to save test scores from SCORM based quizzes

    You can insert pretty nuch any content you like in ATutor test questions, though there will be a step or two more because everything is being done on the Web, as opposed to your desktop.

    If you want to use an external quiz generator, get one that produces QTI quizzes. If you import a QTI quiz, it becomes a part of the ATutor content, and its data gets saved as ATutor data.

    SCORM, AICC etc. are standalone type content, that works on its own.
