    Default Preferences

    I have a few very simple public courses that do not require people to log in. For these courses it is not necessary to have all the menus (Related Topics, Users Online, Glossary Terms, Pop-up Help, etc). However, when users do not log in, they get one of the standard default settings. I want to change the default preferences for this goup of users, so that they will have a clean and very simple user interface. How do I do that? Our ATutor version is 1.4.3

    Thank you.


  • 2005-03-03 18:16:16

    Replace the default prefs

    Save the preference as you want them for yourself, then go into the the members table in the database, and copy the prefs string from your member listing into the atutor_default listing in the theme_settings table.

    Guest users always get the default preference. They only get the course preferecnes if they login.
