    Re: Moving host and upgrading

    The upgrade instruction: http://atutor.ca/atutor/docs/installation.php#upgrade

    You may also like to go through the latest system requirement: http://atutor.ca/atutor/docs/requirements.php

    Another approach is to install a new 2.1.1 and transfer all the courses over using course backup.

    How to backup courses as an admin:
    Go to "courses" tab -> select a course -> click "view" button -> "manage" tab -> click "create" link in the backups section -> follow instructions on the page.
