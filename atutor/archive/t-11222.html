    Re: Video embedding

    Just turn on the Master List feature and it will automatically insert the registration number field (or what ever you end up calling it) on the registration page. This allows a person to create an account on the system. You would not use it on the login screen, though you could have the registration number be the login name.

    Regarding video, in the current version you'd need to use the Object or Embed HTML elements, and write them manually. In the coming releae, only a few days away, you will be able to just click the Insert button next to a movie file, and ATutor will insert the "media" tag, which automatically converts to the appropriate HTML tag used to display that content. Assuming the user has the appropriate plugin or multimedia application installed (e.g. Windows Media Player, Real Player). ATutor, and most other similar systems, do not have there own custom player, though it might be something to add in the future as such players become more readily available..
