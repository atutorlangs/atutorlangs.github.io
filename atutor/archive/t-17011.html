    Re: How to add math expressions?

    In ATutor 1.6.2 you can insert arbitrary math expressions written in TeX, just enclose them within [tex] and [/tex] tags.

    If you save a Word document as HTML for using it in Content you'll have all the formulae converted into images (GIFs) automatically.

    Also there's a visual editor in content; we don't use it much, but there might be such a feature.

    Re: How to add math expressions?

    Try something like the following tool for generating LaTeX equations, then copy the code it generates into an ATutor page, something like this

    [tex]\left(x-1\right)\left(x+3\right)[/tex]

    http://www.codecogs.com/components/equationeditor/equationeditor.php
