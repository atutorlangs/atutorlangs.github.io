    Re: backup courses and migrate to a new install

    Yes you can move a backup from one installation to another, though only the course content, and not the data such as forum threads, links, files etc.

    If you want to backup everything, you would need to backup the MySQL database and the files in the ATutor directory, and move the two onto the new server and restore them there. A tool like phpMyAdmin works wel for backing up and restoring the database.
