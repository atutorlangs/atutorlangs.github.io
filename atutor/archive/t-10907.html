    Re: A tutor and PowerPoint/Flash

    You should have no trouble with Flash or powerpoint. For Flash, use the authoring tool to generate the HTML needed to embed the Flash in a page. Then upload the Flash files into the ATutor File Manager, paste the HTML generated by the authoring tool into an ATutor content page, and adjust the paths to the Flash files in the HTML. There also an example in the demo site, if you login and look for Flash and Object Embedded in the menu

    http://www.atutor.ca/atutor/demo.php

    For ppt file, upload the files into the File Manager, then while using the popup File manager, with the Content Editor, press the insert button beside the ppt files to insert a link into a content page. Students click the link, and open the file into a local installation of Powerpoint.
