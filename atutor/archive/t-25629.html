    Re: Check 94: Issues

    This error is reported because your web page uses <script> html tags. Using <script> tags are totally fine, as long as you make sure when javascript is disabled, your web page is still readable.

    To test it, go to browser setting, disable javascript, reload your web page and verify if your content is still readable and understandable.

    In the case your web page is not readable, re-think about the content structure. You probably wanna use more linear content arrangement.
