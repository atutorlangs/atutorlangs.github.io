    Change front page

    You can add intro test or otherwise change the login page which is what students see when they first visit. That's in the templates folder and named login.tmpl.php
    - make sure you make a copy to put the old one back in case your additiuons stop it working!

    Once they register if they select autologin though they do get a list but only of the courses they've chosen. I agree it would be good if we could choose what we want a student to get when they come in after registration.

    For some the existing My Courses list is obviously great but if you want to create more of a community feeling a default course opening is better. That can act as a reception area and its forum can deal with intro questions about the different courses or how to stuff. You can then very easily add text to this course announcements page to indicate any important changes, like a central News page.

    I'd also like a narrow vertical list of all courses as text links on the opposite side of the existing side menu. Lots of people don't like dropdowns. (In heaven) those the student had signed up for would appear in a different colour and in bold. This might be best as an option for the default course only or others might like it on all pages.

    One day it would be great to have an editor GUI to select which bits to put on the front page.

    which front page

    ATutor doesn't really have a front page (though MyCourses might be considered a front page). You can enter ATutor through a number of different pages. You could also use a course home page as your entry point by linking to it from somewhere outside ATutor.

    e.g. http ://www.courses.com/ATutor/bounce.php?course=23

    Think of ATutor as an addon for a parent Web site, something like the demos are linked into the atutor.ca site. Describe what you need to describe on pages outside ATutor if you can't find a way to do it within ATutor.

    With some inginuity and creativity, just about anything is possible.
