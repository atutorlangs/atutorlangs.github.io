    How to create test questions

    Is it possible to create questions with ATutor?

    In "Creating Test Questions" in the instructors handbook, and it states that "Test questions are created in the Question Database." On the Questions Database page it states that "The Question Database is where course test and survey questions are stored. Questions are created separately.." and goes on to explain how to "Import Questions."

    From this I conclude that I cannot actually create questions within ATutor itself. Is that correct? :(


  • 2010-03-06 13:59:27

    Re: How to create test questions

    Surely you can.

    Test questions are created in the Question Database by selecting desired question type in Create New Question box and clicking Create.

    The Handbook entry was probably intended to mean "separately from tests themselves", because in versions prior to, I think, 1.5 one needed to create a test first and then create questions within that test.
    It may have been put not clearly enough, indeed.
