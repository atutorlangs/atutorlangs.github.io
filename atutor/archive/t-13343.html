    This is a issue with IIS support for PHP, which is lacking somewhat. Though there have been reports of ATutor working with IIS 6, there have been many issues reported. Even if you do get it to work, we recommend Apache instead for production installations of ATutor.

    Apache installs easy with a package like XAMP.
