    Re: Changing Usernames and sending e-mails

    Usernames can not be modified.


    For windows, set the value of "SMTP" in the system's php.ini file to point to another external mail server, perhaps your own personal mail server for testing.

    SMTP could be set to your local server. You'll need to talk to the mail server developers for help with that.
