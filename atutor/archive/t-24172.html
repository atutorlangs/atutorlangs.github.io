    Re: backup restore error

    Firstly, ATutor 1.6.1 is very old, not supported for a couple years now. If you have an opportunity to upgrade, that may fix the problem. Newer versions of PHP do not always work with older versions of ATutor.

    Or, you can check to be sure php is compiled with the zlib library, and check to be sure the content/ and content/backups/ directories are writable. If you're not sure how, your system administrator should be able to help.
