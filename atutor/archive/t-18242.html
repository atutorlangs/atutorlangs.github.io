    Re: Update: Table AT_config creation failed.

    Before you install, edit the install/db/atutor_schema.sql file:

    change


    [php]


    CREATE TABLE `config` (
    `name` CHAR( 30 ) NOT NULL default '',
    `value` TEXT NOT NULL,
    PRIMARY KEY ( `name` )
    ) TYPE = MYISAM;

    [/php]


    To

    [php]


    CREATE TABLE `config` (
    `name` CHAR( 30 ) NOT NULL,
    `value` TEXT NOT NULL,
    PRIMARY KEY ( `name` )
    ) TYPE = MYISAM;

    [/php]


    removing "default ''"
