    Is it possible to restrict an instructor to a specific group. I would like to have certain instructors only view the test results of a group that they would moderate. Is this possible?


  • 2007-08-30 07:42:45

    Re: Instructors, tests and groups

    Yes, if you create an assistant, assign them all privileges the main instructor has except group privileges, then manually add that person to a group like you would add students.

    You can not assign the main instructor to a particular group.
