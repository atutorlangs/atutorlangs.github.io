    Re: I can't install ATutor - No puedo instalar ATutor

    ATutor 2.0 did not have the Javascript check. It was added in 2.1 as a security feature.

    If you have Javascript turned on in your browser (the default) you should have no problem installing ATutor.

    You will need to look through your browser settings to find where javascript is turned on or off. You could try searching Google for that information.
