    Re: Upgrade instruction

    see: http://www.atutor.ca/atutor/docs/installation.php

    in short, here's how I do it:

    1. back up and download my entire database with phpmyadmin or similar tool.

    2. upload the new ATutor directory, normally I name it something like atutor-new

    3. when both the new and old install is on my server I rename my running (old) install to atutor-old or similar, and rename the new upload to reflect my atutor install dir (typically the previous name of the old version).

    4. go to the new dir in my browser, and follow the upgrade instructions.

    Re: Upgrade instruction

    hia,

    no, the db is changed, probably quite a bit from such an old version. you take a backup of it all in case something happens and you need to revert to the old version.

    when you access the folder the new ATutor version resides in, like myserver.td/atutor, ATutor will automatically ask you to do a new install or upgrade, chose upgrade and follow the instructions.

    the db etc are updated by the upgrade program running, and you only need to follow instructions. if anything goes wrong you will still have the old version of the db, as you backed it up, and also the old ATutor files in a different directory.
