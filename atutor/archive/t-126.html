    Content is in the database

    Content created using the Content Editor is stored in the database, rather than in files. Pasted files using the upload feature at the top of the Content Editor are inserted into the Body area, after which they are saved to the db by selecting the Save Content button..


    Files can be uploaded however, using the file manager, which can be linked into your content.
