    Re: Loss of Network Connection

    This is not an issue we've encountered before. There is nothing in ATutor that would cause the session to end if Internet connectivity is lost.

    There is a timeout however (default 20 minutes, set in system preference), though a warning pops up before time lapses, to continue with the current session. If a users does not renew the session, that might cause test results to be lost.
