    Hi, it's under user -> my start page -> Preferences -> Control Settings -> Show Next/Previous Navigation buttons.

    Edited: You can also set the system default by logging in as admin->System Preferences ->Default Preferences ->Control settings -> Show Next/Previous Navigation buttons
