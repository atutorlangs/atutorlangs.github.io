    Re: new language,new module

    addon module language is not part of an ATutor language pack. modules need to be translated separately.

    The language is found in the module.sql file. Copy the English statements into the file a second time, then change the "en" identifier to the identifier for your language, and replace the English in the copied statements with your language.

    Then reinstall the module, or copy the language sql into phpMyAdmin, to install the language into ATutor.

    Send us the translate language when you're done, and we'll include it with the module for distribution
