    Re: cant backup course due to size, and cant find any large files :(

    There's not a verbose setting.

    See what the size of files in the File Manager (not to be confused with File Storage).

    There's not much we can do to help if 2mb is the max file size the server allows. This is a simple setting to change for a provider.

    Yes you can backup things manually via FTP, but without access to the server's shell, you're limited in your options. This would normally be something a sys admin does.
