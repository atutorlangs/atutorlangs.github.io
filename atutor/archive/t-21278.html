    Re: how can chenge the font??

    You can either change fonts in your personal preferences, to set them to your personal preferred font.

    Or. you can modify font styles in the style.css file for the theme you are using to set the default font system wide.

    theme/[theme name]/styles.css

    Users can always override system fonts with their preference settings.
