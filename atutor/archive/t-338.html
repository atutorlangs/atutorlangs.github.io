    Use the Object element

    Use Flash as you would in any Web page, using the HTML Object element. The Flash authoring tool will generate the required HTML, which you would then paste into an ATutor content page. Upload the flash movie to your course's content directory, then point the markup you pasted in, to it.

    Upload power point slides into your course's content directory, then link to them so your stduents can download them for viewing in a viewer, or in power point.

    See the Flash demo in the Play Here Course included with the ATutor demos. Find yourself an HTML tutorial on embedding Flash with HTML if you need more information.
