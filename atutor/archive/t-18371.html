    Re: Not able to view image inserted

    If you are talking about inserting images from a course file manager, the paths to images should be relative to the ATutor root directly. So, if you have a directory called images and the image, myimage.jpg, is in that directory, the path to it would be

    images/myimage.jpg

    Also be sure there are no symbolic link, or shortcuts in the case of windows installation, in the path you defined for your content directory when you inastalled ATutor.
