    You could potentially comment out the safemode check in install/index.php to get ATutor installed, but any functionality in the system that writes to the server would not function. ATutor is not intended to be used with safemode turned on.

    Re: Safe mode

    There are so few systems out there running now in safemode its hard to justify a budget for adding separate FTP support. If someone really wanted it, and wanted to fund its development, we'd add it, but otherwise its not on our list of things to do. It would be much cheaper to find another host that has safemode turned off.

    Safemode is really just for overly paranoid ISPs.
