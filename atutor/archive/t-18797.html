    Re: importing scorm files

    You need to install the SCORM Packages module, and during the install create a directory called "sco" and make it writable by the Web server (i.e. chmod 777 sco)

    The SCORM Package's tool plays SCORM Shareble Content Objects (SCOs). You can import SCORM Content Package directly into the ATutor content.
