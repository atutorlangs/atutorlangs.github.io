    Re: Upgrade ATutor Questions and Request

    1. an upgrade will not change your courses.

    2. it depends on why these features are not working whether an upgrade will fix them. They should work fine without any change to a default php/mysql/apache installation. If a provider turns off some capabilities, like file uploads for instance, an upgrade won't change things.

    3. you can do a fresh install beside the current version, and copy over each course as a course backup (61 times).

    4. An upgrade is the way to go, I think. A backup of the old version gets created, so if anything should go wrong, it is possible to revert. Or, if you don't mind copying all the courses 1-by-1, a fresh install would also work.

    As I recall, you have a custom theme. A 3 ticket pack should cover the upgrade/install and a theme upgrade, though depending on the theme there may be a little more effort involved.

    Re: Upgrade ATutor Questions and Request

    There seems to be something going on on your server if all these things are not working properly.

    1.The gradebook is working fine on our installations without any adjustment. Make sure tests are set to only 1 attempt to have them available to add to the gradebook.

    1b. Once the certificate module has been installed, you need to create the certificate template. You can take the default template and adjust it. There is a little effort involved in creating the template, which is described in the module's README file, linked from the module listing in the admin's module manager. You will need Adobe Acrobat to create your template.

    2. the assignment dropbox relies on File Storage. If FS is not working, the dropbox won't work. Again, it it seems to be an issue on your server.

    3. I'm not sure what you mean by "is there a better way." There'll be some effort involved in creating test questions with any system you might use. You could try something like Respondus for instance, then export from it and import into ATutor.

    AContent is a separate application, not part of an ATutor install.
