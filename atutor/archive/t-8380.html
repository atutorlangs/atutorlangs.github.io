    Re: Uploading Folders

    One file at a time. You can zip files together, upload the zip file, then unzip it with the file manager.

    You may need to have the administrator increase the space (default 10MB), the the size of files that can be uploaded (default 2MB). PHP also puts limits on the size of files that can be uploaded (see the php.ini file)
