    Admin users functions v 133

    Hi,

    We have just installed ATutor version 1.3.3 on a Lynux Red Hat Server, Running Apache and MySQL.

    Under the Admin users menu we see only:

    * Home
    * Users
    * Courses
    * Course Categories
    * Language
    * Server Configuration

    As far as we can see the admin user cannot create users, but can delete them or change their status to instructor or student.
    The admin user cannot create courses, but can delete them or change the properties (tracking on and off, change access details)
    The admin user cannot export or import content packages.

    We were expecting that the admin user may be able to do the following things:

    * create users
    * import users from a CSV file
    * import and export content packages

    Can you confirm whether the admin user can do these things in version 1.3.3 and if so, how do they access these functions.


  • 2004-03-29 12:14:15

    Admins have full control

    Create users using the registration form, or enter a course to import a CSV file of users.

    Import/export content by entering a course and doing it from there.

    There are no Admin screens for this functionality. Admins can access courses as the course instructor to do course related administration. Individual students can be added using the registration screen, as any users would register.
