    Re: I think somebody hacked my site

    It looks like your include/config.inc.php file was left writable after ATutor was installed. Have a look at it and see if there's anything that looks out of place. The only thing that should be in this file is variable settings and comments.

    If editing the config file does not help, talk to your service provider to see if they have a recent backup of your site. Restore the file from the backup. Then remove write permissions from the file. See the installation instructions for details.

    Once you get your system back up, apply all patches, or upgrade to 1.6.3.

    Reinstall the language packs. You can download them from here.
    http://www.atutor.ca/atutor/translate/index.php

    You should still be able to get around ATutor without language by reading the language tokens, which in most cases resemble the actually English language used to replace the tokens.

    You can also unzip a lnaguage pack and manually install the language.sql file using something like phpMyAdmin.
