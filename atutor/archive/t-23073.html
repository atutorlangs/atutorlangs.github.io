  • 2012-05-20 09:28:19

    Re: Some modification

    Both can be done

    1. When creating a folder, available test can be chosen as prerequisite tests, access to the folder only after passing the test. Using the content editor, click on the Tests tab to associate an available test with the page for a followup exam.

    2. Tests results can be exported as CSV. In the test manager select a test with results, choose Statistic, then Submission Statistics, the export CSV
