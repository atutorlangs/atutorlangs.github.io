    Re: Safe mode problem

    You can install ATutor on your local system if you want to try it before commiting to change ISP. It runs on all platforms.

    Install the XAMP or EasyPHP server software, then install ATutor into the web server's root directory (ie. in the Apache part of XAMP or EasyPHP). Links to the server software are at the bottom of the same page you downloaded ATutor from.
