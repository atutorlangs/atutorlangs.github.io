  • 2010-01-06 14:07:19

    Re: Import of SWF scorm

    If you are importing SCORM Content Packages, which are really IMS content packages, then use manage>import-content. If they are SCORM SCOs (sharable content objects) then use the SCORM Packages module.

    I know for sure that importing IMS/SCORM content packages works, so it maybe the tool you are using does not produce valid content packages, which is very common. There's not a lot we can do the help with that.

    My suggestion is to create your content in ATutor itself, rather than creating it in a third party tool. See the content section of the Instructor Handbook for details on creating content with ATutor. Upload your files into the File Manager, then link them into ATutor content pages.
