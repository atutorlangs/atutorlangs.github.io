    Default position of content within the course

    When adding new content to a course... how can I position it?

    What is the position of new added content by default?

    For instance: I got the following course tree:
    1 XXX
    1.1. XXX
    1.2. XXX
    2 XXX
    3 XXX

    If I am located as current position on 1.2 content, I click on add content (I want to have it being created as 1.3) and it goes directly after content "3" as new content "4"...

    How can I avoid having to enter "properties" tab each time I create a new content?

    I couldn't find an answer to this question in the ATutor on-line course neither in the FAQ.

  • 2004-12-23 19:47:52

    Use Add Sub Topic

    From the parent content page (1.0) click on Add Sub Topic in the Editor Bar. This will insert the next available topic number.(1.3 in the case above)
