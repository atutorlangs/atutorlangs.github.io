  • 2005-04-13 09:18:44

    Spec looks good

    The specs for the server look good. I can't see any reason why nothing below the header gets displayed. It's not something we've seen before.

    Double check the config file to be sure you did not add in a trailing space at the end of the file. The closing php tag should be the last characters on the page, without any spaces following. Make sure you are saving the file using Unix end of lines, instead of windows end of lines (see you text editor's options). When saving with windows EOLs, scripts can break when they are run on a Unix based system.

    You could also try manually installing the default theme and see if it works. Copy the file from the default theme into the theme directory, and update the Themes table with an entry for theme, and set it as the default theme (set to 2 in the status column, and the other theme to 1)

    Try installing a separate version of ATutor on this server and see if the same thing happens.

    If these suggestions don't help, I'll try to dig deeper though we will need shell access so we can see and modify the source code if necessary.
