    Re: Do I need to be able to run cron jobs to use Atutor?

    No need to run cron. It's optional. Leave it turn off (the default) if you don't want to use it.

    Apart from custom scripts that might use it, only the ATutor mail queue is currently using it. If you don't use cron, it just means tools that send mail will run a little slower.
