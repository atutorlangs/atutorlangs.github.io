  • 2011-06-03 14:40:28

    Re: Default text size

    You can potentially tweek the stylesheet for the theme being used, adding a font size the the body{} style

    eg. themes/default/styles.css

    eg. font-size:.8em;

    You may need to adjust other measures as well if changing the default font size throws off the layout

    Font sizes are also controllable in users' text preference settings, so it is possible to override the default font size on a user by user basis,
