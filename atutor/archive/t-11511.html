    Re: How to disable public registration in AT 1.5.5?

    Log in as Instructor
    Go to Manage
    Next go to Properties
    Actvate
    Private: Available only to users with an ATutor System Account, and Instructor approved enrollment. Login required. Enrollment is required.

    And ctivate also

    Hide this course from the Browse Courses list.

    If you want to dissable Regristration of new members
    I made it as followed
    Log in as Administrator
    Select System Preferences
    If you enable
    Authenticate Against A Master Student List (Default: Disable)
    and the new registration is not in this list he can not create a account
