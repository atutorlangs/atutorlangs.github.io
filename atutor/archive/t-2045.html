    Import CSV

    Language packs are plain text CSV (comma separated value) files. You should be able to unzip the file and import language.cvs directly into the lang2 table with phpmyadmin.

    Before you try this though, make sure the content/import directory is writable, and try importing via ATutor again. If you still have trouble, then go with the phpmyadmin import CSV.
