    Re: using TILE with arabic

    I am not aware of a LOR available in Arabic. If you do find one, please mention it here so others are aware. Depending on the LOR it may be possible to create a module to link it into ATutor like that done for the TILE and TLE repositories.

    Regarding the character set problems, the current language pack is published in WINDOWS-1256. You might try converting the language pack with the <strong>UTF-8 conversion utility</strong> linked from the atutor.ca translation site.

    http://www.atutor.ca/atutor/translate/index.php

    Before you import the converted pack into ATutor, delete the old one from your installation
