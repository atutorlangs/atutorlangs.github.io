    Re: Importing Languaje

    Well, the problem is described quite well in the message itself.

    You can only import language packs that match your ATutor's version. There's no published Spanish language pack for 1.5.5 yet, and packs for 1.5.4 or any earlier version won't fit.

    If you do want to have Spanish installed anyway (with some language terms missing and a few others incorrect), you should edit your include/vitals.inc.php file and set AT_DEVEL_TRANSLATE to '1'. After that you would be able to import any partial language pack available on atutor.ca.

    Then you could finish the translation yourself if you wish.
