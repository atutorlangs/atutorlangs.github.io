    Re: need help with payment prior to enrollment

    If I understand you correctly, my guess is you need to set your courses to Private, in the Course Properties. The Payment module will then enroll students only after a payment is received. If "Auto Approve" is disabled, then the instructor must manually add students using the Enrollment Manager after a payment is received.

    As for a Pay Now button, typically a separate page is created on a main Web site, to which ATutor has been added, and student follows a link from there to the make payment page in ATutor.
