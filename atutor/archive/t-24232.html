    Re: Course Structure & Student Enrollment

    You would use the admin's Auto Enrollment tool to set this up. Login as the admin, then go to Courses>Auto Enrollment

    You would create three separate enrollments, perhaps with titles like "Line Worker Courses" Choose from the available courses the ones that should be assigned to each enrollment category.

    Once each enrollment has been saved, a link is created that when followed will enroll the registrants in each of the assigned courses. You'd send that like to workers, or perhaps create links on the parent Web site workers could follow to register for their courses

    See the Handbook page on Auto Enrollment, top right while on the Auto Enrollment screen for more details. You can try it out on our demo site. Use the Administrator Demo.

    http://www.atutor.ca/atutor/demo.php

    To setup percent complete, you'd need to install the Certify module. Not sure though if it will serve your purpose. You can see the readme with the module to determine if it does what you need.

    https://github.com/atutor/certify/blob/master/README

    Being Open Source, there's always the option of modifying things to work the way you need them to.
