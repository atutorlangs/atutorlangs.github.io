    Re: How instructors can view the students results?

    don't know if I understand your question right, but I'll try to answer:

    1. N/A means not available (I guess)

    2. if a mark is not available, it is most probably because you have not set any points on the questions, so there's no points there to set a mark based on.

    3. so, to set points on questions, click test & surveys -> [a test] -> questions -> set the points

    4. you will now get the points the student got, not N/A

    5. optionally: go to a [a test] -> edit, and set a pass score in the test prefs.
