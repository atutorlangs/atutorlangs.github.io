    Creating new users with admin

    Hi All

    I have ATutor 1.4.3 installed on a wamp5 webserver. There is no real problem with Atutor itself. But I would like to create new users using the admin login, but I can't seem to find the option anywhere. Is there no other way to create users rather than registering each one from the login screen?


  • 2005-02-15 10:11:50

    Enrollment Manager

    There is no admin tool yet for adding users. Perhaps in the future...

    You can register users one at a time using the registration form while not logged into ATutor.

    or

    You could create an empty course and use the Enrollment Manager to add users to the system. Upload a CSV text file as described there, then enroll the students you've added into the course to send them their login details.
