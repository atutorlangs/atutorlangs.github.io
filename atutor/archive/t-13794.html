    Re: How to show the website by IP address

    You have to have an IP address assign by your network manager if you want to using something other than 127.0.0.1

    When that's done, you don't have to do anything in ATutor, assuming Apache is configured properly. You may need to adjust the value of ServerName in the apache conf/apache/httpd.conf file. ATutor will automatically use the IP address you were assigned.
