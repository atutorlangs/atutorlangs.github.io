    Re: File storage-naming files

    Yours would be a special case. For most having the file storage function like other file manager type applications is most usable. Adding numbers to files would be inconsistent with the way other file managers work.

    If something like this were to be added, it would need to be done in a way that numbering could be turned off. If you want to try creating a numbered display in File Storage, we might consider adding it if it's create in such a way that it does not disrupt the current display (i.e. can be turned off), and build it into a set of patches we could apply to the latest stable version, we might consider adding it.
