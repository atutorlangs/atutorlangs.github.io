    You can manually embed videos in 1.5.4 using the HTML Object or Embed tags, or you can upgrade to 1.5.5 and use the Media tag that was added in that version. See the handbook for that version for more information.

    No, there is not an attendence tool beyond monitoring students activities in a course using the tracking tool.
