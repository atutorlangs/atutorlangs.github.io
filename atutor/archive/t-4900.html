  • 2005-06-22 09:01:05

    move it then edit config.inc.php

    Because the content directory is writable, others could potentially write or upload files to that directory, and get content from course content directories without having to log into the system.

    Move the directory then edit the include/config.inc.php file to point the path defined for AT_CONTENT_DIR to the new location.
