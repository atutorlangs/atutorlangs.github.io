    Re: Missing [Settings] from Packages

    I'm not sure what you mean by "bring it out into a course page"

    Packages (SCORM SCOs) run in their own environment, and don't become a part of the ATutor content. That said however, it is possible to import the content of a SCO, though not the sequencing, into an ATutor content page using the Import feature in the content tools.

    If you mean "how do you make a package available to students", then you need to turn on packages under Manage>>Student Tools.

    The Settings feature was disabled in the current version of the SCORM module, and should not be appearing as a "phantom" link. Ignore it. The Settings features will be brought back in a future version when we've had a chance to improve on it.
