    Reinstall

    ATutor should run on Fedora without any trouble, though using Apache 2 you may have some minor trouble because of its less than stable support for PHP. Apache 1.3* is recommended.

    Try installing again, carefully following the instructions on setting directory and file permissions as mentioned in the install script. The permissions you've mentioned are incorrect.
