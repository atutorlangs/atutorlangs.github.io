    Re: scorm packages from other plattforms to atutor

    Assuming you have the ATutor SCORM module installed, ATutor will import from systems that produce validated SCORM 1.2 packages. Not all systems will produce valid packages. Be aware though that the ATutor SCORM module is no longer being maintained, in favour of importing/exporting using IMS standards.

    Though you can import SCORM content packages into ATutor using the IMS import tools (Manage>Content>Import/Export) it will only import the parts of the SCORM package that conform with IMS standards.
