    Take a test within AContent

    Hi,

    we are using AContent for a project and develop accessible e-learning content with it. As we are also using tests within the lessons, I wonder if it is possible for the learners to do a test within AContent.

    Looking at ATutor, I would say we have to add some php files, e. g. test_intro.php, take_test.php and view_results.php. Is that right? Anything else to add? Or am I just missing something and it is already possible to take tests within AContent and not just developing and previewing tests?

    Kind regards

    Bjrn


  • 2011-10-31 09:08:11

    Re: Take a test within AContent

    No, AContent is not intended to be used to deliver tests. It can be used for authoring tests that will import into an LMS that support the IMS standards.

    ATutor, or some other elearning too would be used to deliver a test created in AContent.
