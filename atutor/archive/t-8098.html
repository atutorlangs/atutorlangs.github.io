    Re: Need to change the module name

    Use the admin's Language Manager, under the System Preferences tab to change the language. You will need to turn on the transalte tools by setting the value of

    AT_DEVEL_TRANSLATE to "1" in the include/vitals.inc.php file.

    You will probably want to adjust the handbook language as well, in the documentation./ directory
