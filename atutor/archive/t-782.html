  • 2003-12-14 11:22:54

    RobotDemo does import into ATutor

    Hello Marco
    I had no trouble importing the RobotDemo into ATutor, however the package contains Director files (.dcr) that would require a Director plugin to display.

    If you are having trouble importing the package onto your system, be sure the upload_max_filesize variable in php.ini is set to a value sufficiently large enough to allow uploading of large files. By default this value is set to 2mb, so it will need to be set to something larger than the 5.1 mb size of the RobotDemo package.

    Also be sure the Max File Size and Course Quota are set large enough to allow upload the file. Use the administrator's course properties tool to reset these sizes.

    Moving content from 1.2 to 1.3+

    You can import content from 1.2 created using the 1.2 "export" tool, into 1.3+ using the 1.3 "backups" tool.

    It is also possible to upgrade your system without any importing/exporting. Your content will remain intact when you upgade.
