    Andreas,

    I don't think you can in the version you are using (1.5.3.2) but in version 1.5.4, the system admin can set the Display Name Format via the System Preferences. The options are:

    Login Name
    First Name Second Name Last Name
    First Name Second Name Last Name (Login Name)
    First Name (Login Name)
    Login Name (First Name)
    Login Name (First Name Second Name Last Name)
