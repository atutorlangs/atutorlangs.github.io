    Uploadd user list from enrollment manager

    You can upload users to the system and enroll them in a course using the enrollment manager. (tools->enrollment manager->import course list)

    You need to create a csv (comma seperated values) list of users. Or you can manually add users by typing them in 5 at a time.

    Hope this helps.

    P.S. there is documentation for this in the howto
