    Re: security in ATutor

    See the following thread:
    http://atutor.ca/view/7/4732/1.html

    The recent minor security issues identified in 1.5RC1, regarding form input validation in a few places, have been dealt with and will be available in the 1.5 stable release.

    You will want to implement an SSL based installation of ATutor for full security.

    Being GPL software we can make no gaurantees outside a contracted agreement. We do however make every effort to ensure security.
