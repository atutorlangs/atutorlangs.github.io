    Re: Mathematical equation on Atutor

    It will likely be a mathml utility we add for mathematical notation, mainly because output has the potential to be accessibile to assistive technology users. Output from tools such as latexrender are images, which will not be accessible.

    We have experimented a little with mathml. but a math notation authoring tool is fairly low on our todo list. Someone else might take on the task of creating one, or creating a latexrender module for ATutor. Creating a module would probably not be all that difficult. Here's how it got done for the b2evolution blog, if someone wanted to use this example as a model for creating such a plugin for ATutor.

    http://www.vyvy.org/blogs/index.php/2005/09/22/latexrender_plugin_for_b2evolution
