    Re: i m not able to install atutor

    Try
    http://127.0.0.1/path_to_atutor/ATutor/index.php

    If that works, then you need to add "index.php" to the DirectoryIndex line in your apache httpd.conf file, like so:

    DirectoryIndex index.php index.html

    Then restart your webserver.
