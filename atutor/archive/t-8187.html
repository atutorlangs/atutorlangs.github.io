    Re: How much hosting do I need?

    5 courses with 50 students will run on any ISP. That would be a very small ATutor installation.

    The space you need will depend on what you plan on presenting as content. Movies for example, take up a lot of space. Text and image based courses take up little space. If the latter, 25mb per course should be more than enough (5 to10MB/course will probably do).

    See the system requirements for details about supporting software that needs to be present. Nowadays all ISPs have this software installed. You just need to make sure the versions are sufficient.

    http://atutor.ca/atutor/docs/requirements.php
