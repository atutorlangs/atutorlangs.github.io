    Try Apache

    Try using the Apache webserver, or install EasyPHP (which includes Apache). You can download EasyPHP from the downloads page below. It should install with a few clicks.

    http://www.atutor.ca/atutor/download.php

    Php was created for Apache, and works well with it. Php support in IIS is less than perfect. Reports suggest ATutor works for some installations of IIS, but not others.

    Unfortunately we do not have a 2003/IIS system to test with. If you discover solutions, please post them here.

    Any EasyPHP

    Any version of EasyPHP will do that has Apache 1.3* included with it.

    You can keep IIS. You will need to run either Apache or IIS on something other than the default port 80. See the configuration options in either to change ports.

    http://www.easyphp.org/
