    Re: Installation on Vista

    This is a vista problem with read-only files.

    Try doing this:

    1.. Find the folder you need access to from an explorer window
    2.. Right-click it
    3.. Click Properties
    4.. Click the Security Tab
    5.. Click Edit
    6.. Click Add
    7.. If you are the only one that needs modify access to this file, type
    your username and press enter. Otherwise, type "Users" and press enter.
    8.. Click the checkbox under Allow next to Full Control
    9.. Click OK
    10.. Click OK
    NOTE: This will work on folders that you have created or folders that directly contain your files, such as My Documents, but will not work on system folders such as Windows, Program Files, or Documents and Settings.


    ref: Windows Vista Support Faq
