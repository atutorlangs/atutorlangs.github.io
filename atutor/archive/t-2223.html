    Please help yourself

    Things like this you should try to discover yourself. The free help offered here is intended to address technical issues. There's documentation and demos available so you can conduct your own investigation.

    The ACollab contact list is group specific. It is a list of people who are members within a single group, which when run as an ATutor addon, gets it information from the ATutor members table. You could create a single group with all members of an ATutor course as members of a single ACollab group.
