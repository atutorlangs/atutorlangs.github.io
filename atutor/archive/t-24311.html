    Re: Double Quotes after upgrade

    The double quotes issue was fixed in patch 5 for ATutor 2.0.3 (see http://atutor.ca/view/7/22854/1.html for more details).

    I'm guessing this issue occurred for me because of custom theme files during an upgrade from 1.6.2 to 2.1.1. I was able to use a modified version of the patch file located here (http://update.atutor.ca/patch/2_0_3/2_0_3-5/patch.xml) to patch my custom theme files.

    This is likely to be the same issue that causes images to not display in questions which other users have posted about in these forums.
