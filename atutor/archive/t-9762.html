    Re: ATutor freezing

    It's unlikely to have anything to do with ATutor.

    It could be one of any number of problems from a failing disk, to network slowdowns, to a misconfigured web server or database...

    Examine your system's errors logs, and the ATutor admin's System Preferences>Error logging to see if you can find any clues there.

    You could try installing it on another system, preferably set up with the same php/apache/mysql combo to help rule out server software problems.
