    Re: SSO between atutor and other system

    There is a third party LDAP module, though it has not been supported for some time. There's also an OpenID module for logging in with various facebook, google, twitter etc. account. We're currently looking at extending that module with CAS support.

    If you are working with ATutor 2.2, you can pull the latest version of the OpenID module from github. Otherwise try the one available through the Modules download page.

    http://www.atutor.ca/atutor/modules/
