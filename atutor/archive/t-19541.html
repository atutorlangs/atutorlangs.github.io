    Re: 1 on 1 conversation between a student & Instructor

    No, there isn't a way currently to disable a forum. Might be a good feature for us to add though.

    You could use the Export Forum tool to package up a forum, then delete the forum. The file in the archive is a a bunch of html files. If you wanted to make the archive available for others to see, upload it into the file manager, unzip it, then link the index page into your course somewhere.

    Search the handbook is you need more details.
