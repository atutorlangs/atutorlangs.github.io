    Re: how make Atutor based google searchable

    With some effort you could make ATutor searchable using the Apache ForceType directive: See the Apache documentation for details.

    Here's one of many tutorials on using ForceType found on the Web:

    http://www.opensourcetutorials.com/tutorials/Server-Side-Coding/PHP/friendly-urls/page1.html

    If you do this, google will be able to spider only content in public courses, since google can not otherwise login to protected and private courses.
