  • 2008-07-02 08:13:31

    Re: Test without end dates?

    Creating tests without the end date is impossible.
    You can set the date to 3 years ahead though, which has pretty the same result in practice.

    The removal of "leave blank" options has been discussed a few times here, and you can find a fix for version 1.5.5 along with instructions here: http://www.atutor.ca/view/7/12303/1.html .
    I didn't study the changes in that area in 1.6+ code yet, but the code modification required should be minimal, if any.

    It would be nice to have this in a patch for the new Patcher module. Unfortunately, I don't have time to do that job for now.
