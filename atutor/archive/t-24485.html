  • 2013-11-29 09:43:52

    Re: link Atutor with a website

    ATutor will install fine on an Ubuntu system. You can install it as a sub-directory in your current site. You don't need a separate domain. To link it back to your main site, when setting up ATutor, or editing the site's System Preferences, enter the location of your parent site in the Home URL field.

    See the installation instructions for additional details:
    http://atutor.ca/atutor/docs/installation.php

    Re: link Atutor with a website

    Assuming ATutor is running correctly on your local server, you'd need to find out the IP address of the machine, and assuming your firewall allows access to port 80, you would link to ATutor with something like http://123.234.345.456/ATutor, changing the IP address of course to your own.

    This is probably not the right place to find help on setting up a Web accessible server. There are lots of resources out there you can find through Google etc.
