    Re: Installation issue

    You probably won't get ATutor working with IIS 5.1. It's too old. You can upgrade to 6.0+, or we recommend using Apache instead. Use something like XAMP or EasyPHP to setup the system (they both install Apache, MySQL, PHP, and phpMyAdmin), then unpack ATutor into the Apache web document root (htdocs/), and open it in your web browser to install it.

    The XAMP download can be found near the bottom of the downloads page:

    http://atutor.ca/atutor/download.php
