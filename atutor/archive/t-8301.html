    Lectora produces very broken HTML markup. If you copy the HTML source from a Lectora content frame and pass it through the W3C validator, you will see what I mean.

    http://validator.w3.org/

    There are other issues with using Lectora content, one being it forces users to view the content in a framed environment. ATutor does not use Frames. While you *can not* view Lectora content as imported "content", you should be able to view the content as an imported "package" using the Packages tools. But because the markup is broken, it does not display properly.

    We recommend using Reload as a SCORM authoring tool
    http://www.reload.ac.uk/editor.html
