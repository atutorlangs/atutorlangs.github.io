    Re: Backup does not include videos, mp3 nor list of user names and e-mails.

    * How to upload files into file manager:

    Login as instructor -> "Manage" tab -> "File Manager" link, where you can upload a file, create a new file via ATutor interface, or create folders for organization.

    * How to embed video or mp3 into content:

    At pages "create content" or "edit content" -> access "file manager" link (version in or below 1.6.3) or icon (in version 1.6.4+) -> find the file you wanna embed -> click "insert" button besides the file.

    BTW, the forums is the usual means that ATutor users ask questions because the Q & A would benefit the whole community.

    The bounce-back of the emails to info@atutor might because of our recent office move. We will check on that.

  • 2010-11-10 15:07:20

    Re: Backup does not include videos, mp3 nor list of user names and e-mails.

    You might have known that students' names and emails can be found at "manage" tab -> "enrolment" link. But enrolment is not included in the backup. The backup is more for keeping the information that can be shared across courses, for instance, banner, content, assignments, tests, files managed and used by the course instructor, etc. The course-specific info like enrolment is not included.

    Moreover, the super admin account is created in the process of ATutor installation.
