    I can't reproduce this in 1.6.1. Random tests seem to be working fine. I created a test with 50 questions, and randomly selected 3 questions to be displayed. 3 questions were displayed.

    Be sure questions have the same mark value, and that no changes are made to the test after it has been taken once or more. Once the test is taken, it can not be changed, otherwise the statistics generated for the test would be skewed.
