    Link directly to a course page

    If you are asking for help, provide lots of detail so problems can be reproduced.

    It is possible to link directly to the Home page of any course: for example my Search Engine Marketing Course URL is:
    http://www.chin.gc.ca/ATutor/bounce.php?course=37

    SADLY, we cannot link directly to any next level page in the content navigation, for example, this URL is the author pageof my Search Engine Marketing course :
    http://www.chin.gc.ca/ATutor/content.php?cid=996
    If you enter this in Explorer, you will land on the login Page instead of the Author page. The course does not required any login and password.

    MY QUESTION
    How can I link directly to a second or... level page (any pages) of my course?



    Things to describe:
    Operating system ATutor is installed on -
    MS Window XP

    Re: Link directly to a course page

    If you are using 1.6.1 or above, you need to turn on Pretty URLs in System Preferences to accomplish what you have described. See the Handbook for details.

    Prior to 1.6.1, you can not link directly to pages in courses.
