    There is no German yet

    You have to start with English as the Default language (en), then download the German language pack, install it, then set the language to default to German (de).

    The German language pack is not yet complete. You might volunteer to help complete the language by requesting a translator access through My ATutor. You will need to be a registered ATutor member before you can access the translation tools on the ATutor.ca Web site.

    Or, set the default to es if you are using the Spanish Language pack.
