    [green] Question.

    Is that a known issue for Content Package export/import? Should i have used the Common Cartridge export?[/green]

    I've not experienced missing images with Content Packages or Common Cartridges. Look at the paths to images in the original content to see if there are any patterns there.

    Common Cartridges are more common now than Content Packages, though both work equally well for moving content. Common Cartridges also allow you to associated tests and forums with you content, so they get imported/exported with your cartidges.


    [green]Questions.

    Should categories have been transferred along with the test items? Is there way to make that happen? [/green]

    If you were using the Export feature in the the Test Manager, it exports tests as QTI packages, a standard from the same people that created Common Cartridge and Content Package stardard. The QTI standard does not imclude question categoried.

    If you want to keep the categories, take a Backup of the orginal course using the Backups Manager. Upload a backup into the new course and restore just the tests. You can also creat a new course and choose the backup created above, as the initial content, selected near the bottom of the course properties set when creating a course.
