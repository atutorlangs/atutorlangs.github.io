    ATutor 223 for PHP7

    ATutor 2.2.3 now works with PHP7, otherwise the same as ATutor 2.2.2 with a few minor bug fixes.

    A beta version is now available at:
    http://www.atutor.ca/atutor/download.php

    This version can be used to create a new ATtuor installation, but should not be used to upgrade an existing version of ATutor.
