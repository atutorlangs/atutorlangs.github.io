    Re: Atutor Theme - Force One Theme

    Not sure how a theme set on one site could override one on another site. It should not be possible. Maybe the student has their theme set in his/her preferences.

    The way around it is to disable all the themes except the one you want your site to use. Admin>System Preferences>Themes.
