    Old LRN

    The manifest uses an element called tableofcontents, which is no longer a part of the IMS specification. It was phased out in v.092 of the spec in the year 2000. The current spec is 1.1.3.

    You will need to recreate your content in a package editor that uses V1.0 of the specification or higher. See if there is a newer version of LRN that might take old LRN packages and update them to the current specification.

    For future reference, can you mention the version of LRN you are using.
