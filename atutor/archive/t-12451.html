    Re: Installation

    Your webserver is defaulting to index.html instead of index.php. This can be adjusted in the webserver's configuration file. I'll guess you are using Apache, in which case you would add index.php to the values for DirectoryIndex

    [php]

    DirectoryIndex index.php index.html

    [/php]

    Re: Installation

    We would not attempt an install on a Windows system. Support for PHP is just too poor for us to offer any kind of gaurantee.

    If you want ATutor installed on your windows system, you are best to use a package like XAMP, which includes Apache, MySQL, and PHP. See the downloads page for a link to XAMP. The other possiblity is to install ATutor2Go, which includes everything.
