    Re: How can set My language as default

    There's a bug in the 1.6.3 language manager that causes the no languages found error, which has since been fixed. You could try replacing the file with a fresh copy from the code repository to see if that helps.

    http://atutorsvn.atrc.utoronto.ca/repos/atutor/trunk/docs/include/classes/Language/LanguageManager.class.php

    Once you've replaced the file, you should be able to delete English. There is no disable. My mistake.

    FYI, there are three levels off language settings. The default system setting, the course setting for displaying courses in a particular language, and the user setting. The latter settings override the former settings.
