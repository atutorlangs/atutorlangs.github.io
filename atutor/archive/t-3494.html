    Re: classes, forums and beginning test

    I'm sure sure I understand you.

    Yes you can divide users into different classes. Create a copy of your course with the Backup tool, and restore it into a new course.

    By default class do have different forums.

    There is not yet a pretest feature. It is on the list of features coming in a future release, likely June/July 2005.

    Please see the HowTo documentation for answers to questions like these.

    http://www.atutor.ca/atutor/docs/howto.php
