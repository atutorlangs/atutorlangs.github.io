    Re: Loses theme wonce a user logs in, also lost theme administration tools.

    You should probably upgrade to the current version of ATutor, though you will need the recreate your theme. In the current version you can export themes and reuse them after upgrading. With 1.4.3 this isn't possible.

    In your case I'm guessing the user's preference setting is set to display the ATutor default theme. In the current version of ATutor you can set which theme displays by default, though students can still override that selection by choosing a different theme in the preference settings, unless you remove the other themes.

    Like I said, "it isn't possible in 1.4.3".

    If the tools were there at one time, then it sounds like you downgraded from 1.5 to 1.4.3.
