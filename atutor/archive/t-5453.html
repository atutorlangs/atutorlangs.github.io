  • 2005-08-16 16:50:36

    Customizations

    It is unlikely you will be able to import your customized ATutor database into the new 1.5, at least not without some effort. You would need to study each of the upgrade SQL files to see what has changed since 1.4 (and lots has changed).There's not much help we can offer for upgrades to customized versions of ATutor.

    Depending on the customizations you made, we may be able to include them with the public source code so you don't have this problem the next time you wish to upgrade, if you want to share your features. Ideally in the future, where you do customize features, separate them from the main ATutor installation, in their won directory and database tables, then when you upgrade it will be easier to copy over the customisations as modular units.
