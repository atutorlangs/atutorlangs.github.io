    Creating content directory

    Using windows explorer go create the directory that you want to be the content directory.
    eg. c:\\wwwroot\\atutor\\content\\
    Then in step 5 simply put that path in the text box.

    I also found that it gave an error saying the folder didn\'t exist, but when I created it manually then it worked fine.

    Hope that helps.
