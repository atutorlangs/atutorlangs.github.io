    Re: Installing patch without internet connection

    The patcher is a network tool, not really intended to be used without access to the update server.

    You can view the patches and manually download them from

    http://update.atutor.ca/patch/1_6_2/

    Then zip the files together then apply them through the upload field in the patcher.

    Re: Installing patch without internet connection

    Done.

    Here is the procedure I followed, just in case someone else can be interested in applying patches off-line.

    Download all the patches using:

    wget -r -np http://update.atutor.ca/patch/1_6_2/

    now build all the .zip that need to be passed to the patcher module using the attached Perl script (it worked for me, use it at your own risk).

    It's been written on a Windows system but it should be easily ported to a Linux/Unix system.
