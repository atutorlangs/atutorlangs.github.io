    Re: Students to choose course language

    There is not currently a feature that would allow you to change the language of the content. You would either need to create a copy of the course, and present two different courses, or create duplicates of each page. in each language, within one course, and provide access to to both languages for all students.

    If you have a developer who knows some php, a feature like this would not be too difficult to create.
