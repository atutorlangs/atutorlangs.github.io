    Course Themes Not Changing

    I just upgraded our server to version 2.0.2 . Upgrade went fine. I changed the theme back to Atutor 1.6 and while the admin area is changed the courses are not. I login and the theme is 1.6 but as soon as I go into a course it changes the theme to the new Atutor 2 default theme. Ideas?


  • 2011-05-16 08:44:28

    Re: Course Themes Not Changing

    A user's personal preferences theme choice, and course properties theme choice will override the system default theme.


  • 2011-05-16 08:49:53

    Re: Course Themes Not Changing

    Where do I change that. Can I set a default for all users?


  • 2011-05-16 08:55:46

    Re: Course Themes Not Changing

    Change them in My Start > Preference setting, or in a course via Manage>Properties

    You can only set a default for new users. User can always override the system theme setting, unless you turn off all the other themes
