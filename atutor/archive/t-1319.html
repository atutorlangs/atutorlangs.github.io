    Set default lang to English

    I'm guessing you've set the initial default language to something other than English when you installed. In include/config.inc.php set the value of DEFAULT_LANGUAGE to "en". After you have ATutor running correctly, then import other languages and reset the default language to one of those languages.

    If that does not fix the problem, we'll need more information. See "Help us Help You", the first message thread in this forum.

    What is the default language set in your Web browser?
