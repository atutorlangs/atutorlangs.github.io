    Re: Quick Question Regarding Templates

    I think you mean the language for the form labels.

    As an admin, all language is managed under System Preferences>Languages.

    Use the Editor in the Language Manager to create custom language.

    Or, turn on translation, then use the translation tools to search for the term "Sex", and change it.

    See the section Language in administrator handbook, and the various language related sections in the developer
