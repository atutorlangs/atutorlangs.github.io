    Re: File Manager, File Storage and Uploading Assignments

    Upload size and upload total are controlled by the same setting that controls the size and amount of files that an instructor can upload into a course, set by an admin on a course by course basis. Upload size is also controlled by the system's php.ini setting upload_max_filesize.

    There isn't currently a way for instructors to delete files uploaded by students. This would be a good feature to add to our todo list.

    http://atutor.ca/atutor/mantis/view.php?id=4677

    For now if you want to remove a file, or rather the reference to a file, you'd need to do it manually by removing its listing in the database. Not ideal I know.
