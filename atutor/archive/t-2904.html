    Run the installer

    There is an installer included with ATutor and one for apache/php/mysql (EasyPHP).

    We have not had any problems setting up ATutor on XP systems. My only guess is you have mistyped your mysql access info, or the user you are using to connect to mysql does not have create db/table permissions. See the requirements section for more about mysql permissions.

    http://atutor.ca/atutor/docs/requirements.php
