    Re: Payment via Paypal

    I'm not sure how it could be possible to enable a module that has not been installed. You can only enable from the installed modules screen (the module manager main screen).

    We're continually having to tweak the PayPal code in the module as they change things regularly. Unfortunately we don't have the resources to resolve this now. As much as we'd like to support PayPal, its been something we've providing without funds to maintain it, generating a loss for us (while generating revenues for those who use it). Until we find a group willing to support PayPal development, we probably will not continue to support it ourselves.

    I suggest you try MiraPay, which has remained stable for many years (It's the service we use).

    Or, I would encourage you to try to track down the problem yourself, and if you resolve it, send the resolution on to us to add to the module for future versions.
