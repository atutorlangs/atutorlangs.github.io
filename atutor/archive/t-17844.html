    Re: Transferring demographic questions to other courses.

    the easiest way is to create and restore a backup. go to manage -> backups -> create (and download). then manage -> backups -> upload in the new course.

    in most cases you only wish to restore test & surveys from the backup, and keep an eye on overwrite / append.

  • 2009-07-01 09:44:20

    Re: Transferring demographic questions to other courses.

    The upgrade instructions is the best place to start.
    http://www.atutor.ca/atutor/docs/installation.php

    If you are going to upgrade, wait a few weeks if you can. Version 1.6.3 is not far away.

    Read through the updates, and if there is anything that sounds familiar, install the corresponding patch. Backups of the updated files are created so you can revert from an update in the unlikely event something goes wrong, though you do need to know your way around ATutor database and the server. In any case you should install any security patches. The others you can probably leave if you're not experiencing any problems. If you upgrade, don't bother with the patches. They will all be included with the new version.
