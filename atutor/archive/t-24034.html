    Re: My Tests and Surveys

    You can using the language editor to change language. As the admin go to

    System Preferences>Languages>

    You would need to manually remove the leave blank option from the MC template. See:

    https://github.com/atutor/ATutor/blob/master/themes/default/test_questions/multichoice.tmpl.php

    However, you will loose that change when you upgrade. You could create a custom theme with that template in it, along with the other required files for a theme. Reinstall your custom theme after upgrading to reapply the change. See the theme designer documentation in the ATutor Handbook
