    ATutor is multi-lingual

    1. You can have any number of languages installed on an ATutor system. See the language documentation in the Administrator section of the ATutor HowTo 1.4.3 course.

    http://www.atutor.ca/atutor/docs/howto.php

    2. Once the language pack is installed, it can be modified through the same tool used to import the language packs above after turning on translation in the vitial.inc.php file. Finding all the instances of instructor and student might take a little effort, but it could be done.

    Auto select language

    ATutor first detects the language setting in the users browser, and if available in ATutor, displays in that language when the user views the site, either the login, or other public pages. If the language is not available it will default to the language set in the config.inc.php file. Thirdly it will default to the language a users has set as their default language in their profile.

    There is no way to retrieve a users ATutor language preference until they have logged in. Autologin would simulate applying the user's default language.
