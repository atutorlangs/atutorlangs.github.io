    Certify Compatible Friendly Host Found

    I hope I am posting this in the right place, and before you ask I am not an affiliate or employee of either of the companies named in my post.

    I was with Siteground for my hosting, they offer a package which they state - Quoted from http://www.siteground.com/atutor-hosting.htm

    'ATutor is very adaptive and accessible web-based open-source Learning Management System (LMS or LCMS). For your ATutor website SiteGround will provide you with a reliable, secure and fast hosting solution where you can use all ATutor features.'

    All features? definitely not. I asked them to install pdtfk for the Certify Module, they cannot do this as it involves server downtime. I know myself from several test servers of my own that it involves 0 downtime. Go figure!

    After searching around I came across www.arvixe.com, I asked 3 members of support if I switched to them would they install pdtfk, all 3 answers 'Yes this should be no problem as it can be done without server downtime'

    I have spent 3 days switching my site and tested the pdftk install about 10 minutes ago...works perfectly.

    If anyone is seeking a host which supports 'All features of ATutor' consider arvixe.com

    I could not find a way to install pdftk from the CPanel at this host but it took only an hour to upload and install the application manually. I'm sure if you needed an installation the staff would be happy to oblige.
