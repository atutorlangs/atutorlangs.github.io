    Re: Side menu move from the right side to left side

    One way to do it would be as follows:
    Use Firefox 2.x
    Install the extension "Web Developer" for Firefox.
    Using Firefox in the Web Developer extension choose: Information->Show Div order
    In same extension choose: CSS->Edit CSS
    Now you can edit the CSS a little easier :-)
    I tried it and attach a quick and dirty totally UNTESTED left menu version of the bluemin theme as example.
