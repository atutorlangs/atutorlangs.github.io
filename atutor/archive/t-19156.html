    Re: Gradebook

    First, you should not need to install the Gradebook with 1.6.3. It's already installed. You just need to add it to a course via Manage>Course Tools.

    Gradebook is a course tool rather than a profile tool. Once turned on, students access it via a nav tab at the top of a course, or a tool icon on the home page or Student tools page, depending on where you want it accessed from.
