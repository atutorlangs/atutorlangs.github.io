    Yes, there does seem to be a bug. I've added it to the bug tracker. For now try deleting the english language, then setup your system to use another langauge. Once that is working, reinstall English, if you like.

    Here is the priority (that should be) set on language displays:

    Public pages:
    -Previously set cookie
    -Browser setting
    -System preference

    After logging in:
    -User language preference
    -Previously set cookie
    -Course language setting
    -System preference
    -Browser setting
