    Content packages with javascript are generally SCOs, which need to be imported using the Packages tool, rather than importing directly into an ATutor content page.

    One thing we are planning to add is a means of storing any information in the HEAD for an imported content page (including JS and CSS), so it can be included when the page is displayed.
