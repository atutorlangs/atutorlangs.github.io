  • 2007-02-21 14:47:24

    Re: Session expiry

    In "include/vitals.inc.php", line 77, I modified the code as follows:

    @ini_set('session.cache_expire', '180'); /* 3 hours */
    @ini_set('session.gc_maxlifetime', '3600'); /* 1 hours */
    @session_save_path(AT_CONTENT_DIR . 'save_path');
