    There is another way as I described above.

    Your password has not change, but they were encrypted in 1.6, which requires a script in the login screen to interpret the encrypted passwords. The theme you are using is missing the script. If you run the patcher, it will insert the script for the theme.

    If you upgrade, your data all stays the same. There's a reminder on the login screen submenu.
