Translation

Starting in ATutor 2.2.3, languages listed here can be imported from GitHub using the Import tool in the ATutor Administrator's Language Manager. But, if for instance, a firewal prevents accessing this list, then download the zipped language pack from here and upload it through the Language Manager. As the admin of your ATutor installation, in System Preferences open Languages>Import. Use the Browse button to find the language pack you downloaded, then press import.
